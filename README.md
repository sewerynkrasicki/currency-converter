# Currency-converter
Real time currency converter that use API for current exchange rate. Gui is built using PyQt5.

# Prerequisites
- Just clone my repository and replace 'YOUR-API-KEY' with your api key

### Built with
- [Python](https://www.python.org/) - Python Programming Language
- [PyQt5](https://www.riverbankcomputing.com/software/pyqt/intro) - PyQt5
- [CurrencyConverterApi](https://free.currencyconverterapi.com/) - Free Currency Converter API
